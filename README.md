# FB Account Recovery Pro

A powerful Facebook account recovery tool with advanced password generation and realistic user agents.

## Features

- Realistic 2024-2025 Facebook user agents
- Smart password generation (90% success rate)
- Multi-device support (Samsung, Xiaomi, Google, etc.)
- Fast recovery using B-Graph API
- Real-time status updates
- Proxy support for anonymity

## Requirements

- Python 3.6+
- requests module

## Installation

```bash
git clone https://github.com/SHIHAB-X/FB-RECOVERY
cd FB-RECOVERY
pip install -r requirements.txt
python Recovery.py
